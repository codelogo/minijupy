# Jupyter notebook in a container

## Description

The purpose of the project is to be used as a starting template for creating projects shipped in containers and controlling them throught Windows small scripts:
  - docker_build.bat: build the image
  - docker_jupyter_notebook.bat : run a container that executes **Jupyter notebook**
  - docker_bash.bat : run a container that executes bash (instead of Jupyter)
  - docker_exec.bat : enter a running container
  - docker_clean_up.bat : remove the pending containers
